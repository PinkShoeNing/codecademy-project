HTML Tags

Tags for making lists

Tag Name Discription
<ul> unordered list defines an unordered list
<ol> ordered list defines an ordered list
<li> list item defines a list item
<dl> description list defines a description list
<dt> description list term defines a term in a description list

Tags for making tables

Tag Name Discription
<table> table defines a table
<th> table header cell defines a header cell in a table
<tr> table row defines a row in a table
<td> table data defines a cell in a table
<caption> table caption defines a table caption

Elements for HTML forms

Tag Discription
<form> defines an HTML form for user input
<input> defines an input control
<textarea> defines a multiline input control
<label> defines a label for an <input> element
<fieldset> groups related elements in a form
<legend> defines a caption for a <fieldset> element
<select> defines a drop-down list
<optgroup> defines a group of related options in a drop-down list
<option> defines an option in a drop-down list
<button> defines a clickable button
<datalist> specifies a list of pre-defined options for input controls
<output> defines the result of a calculation
